# 🌐 CloudExify — Personal Portfolio

A modern, fully responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript. Designed to showcase projects, skills, and experience with a sleek dark/light theme, smooth animations, and a clean UI.

**🔗 Live Demo:** [mtncloudexifyproject1.vercel.app](https://mtncloudexifyproject1.vercel.app)



## ✨ Features

* **Dark / Light Theme Toggle** — Persistent theme switcher with smooth transitions
* **Animated Hero Section** — Typewriter effect with dynamic role cycling
* **Scroll Reveal Animations** — Elements animate into view as you scroll
* **Skill Progress Bars** — Animated bars that fill on scroll trigger
* **Project Filter** — Filter projects by category (Frontend, AI, Full-Stack, App)
* **Contact Form** — Client-side validated form with success state feedback
* **Page Loader** — Smooth loading screen on entry
* **Fully Responsive** — Mobile-first layout with hamburger nav for small screens
* **Accessible** — ARIA labels, semantic HTML, keyboard-navigable

## 🚀 Getting Started

No build tools or dependencies required — just open it in a browser.

```bash
# Clone the repository
git clone https://github.com/MTahaNadeem/CloudExify-Project-1.git

# Navigate into the folder
cd CloudExify-Project-1

# Open directly in your browser
open index.html
```

Or use the **Live Server** extension in VS Code for hot reload during development.
